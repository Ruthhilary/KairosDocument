# Outreach Programs

The Outreach Programs page is where evangelism events are created and managed. Programs are branch-scoped — members can only register for programs within their own branch.


## Program list

Programs are shown in a table with:

| Column | Description |
|--------|-------------|
| Program Name | Name of the evangelism event |
| Date | When the program runs |
| Location | Where it takes place |
| Coordinator | The member responsible |
| Souls Reached | How many souls were captured at this event |
| Registered | Number of workers who signed up |
| Status | Active or Completed |
| Registration | Open or Closed |
| Actions | Edit or view program details |

Use the **Active Programs** and **Past Programs** tabs to switch between current and historical events. Search by program name using the search bar.

## Creating a program

Click **+ Create Program** (Leader, Pastor, and Admin):

| Field | Notes |
|-------|-------|
| Program name | Required |
| Branch | Required — determines who can register |
| Date | Required |
| Location | Required |
| Address / City | Optional |
| Description | Optional |
| Coordinator | Link to a member — the person responsible |

## Registering as a worker

Members can register themselves as workers on any active program within their branch. Once registered, they appear in the program's worker list and can see their upcoming outreach assignments from their profile.

## Capturing souls

After a program, coordinators and leaders log the souls reached by linking soul capture records to the program. Each soul captured at the event appears in the [Souls Pipeline](/platform/souls-dashboard) and contributes to the **Souls Reached** count.

## Who can manage outreach

| Action | Member | Leader | Pastor | Admin |
|--------|--------|--------|--------|-------|
| View programs | ✅ (own branch) | ✅ | ✅ | ✅ |
| Register as worker | ✅ (own branch) | ✅ | ✅ | ✅ |
| Create programs | ❌ | ✅ (programs they coordinate) | ✅ | ✅ |
| Update programs | ❌ | ✅ (own programs) | ✅ | ✅ |
| View all worker registrations | ❌ | ✅ (own programs) | ✅ | ✅ |
